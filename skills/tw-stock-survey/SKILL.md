# 台股研究調查 Skill

提供台灣股票市場的系統性研究框架。

## 重要聲明

**本 skill 僅提供資訊整理，不構成投資建議。投資決策請自行評估風險。**

## 研究面向

| 面向 | 說明 | 參考文件 |
|------|------|---------|
| 基本面 | 財報、EPS、營收、獲利能力 | [references/fundamental.md](references/fundamental.md) |

## 標準研究流程

### 1. 個股快速掃描

```
輸入：股票代號或名稱
輸出：一頁式摘要報告
```

執行步驟：
1. WebSearch 搜尋「{股票代號} 股票 基本資料」
2. WebSearch 搜尋「{股票代號} 近期新聞」
3. 整理成結構化報告

### 2. 深度研究報告

```
輸入：股票代號 + 研究目的
輸出：完整分析報告
```

執行步驟：
1. 基本面：營收、EPS、毛利率、ROE
2. 產業地位：競爭對手、市佔率
3. 成長動能：在手訂單、新產品、擴廠計畫
4. 風險因素：產業週期、客戶集中度、匯率

### 3. 申購評估

```
輸入：股票代號（即將上市櫃）
輸出：申購資訊與價差預估
```

執行步驟：
1. 搜尋承銷價、申購期間、抽籤日
2. 搜尋興櫃/市場參考價
3. 計算預估價差與報酬率
4. 整理公司基本面亮點

## 常用資料來源

| 來源 | 用途 |
|------|------|
| 公開資訊觀測站 | 財報、重大訊息 |
| 證交所/櫃買中心 | 申購公告、交易資訊 |
| 財經新聞（經濟日報、工商時報）| 產業動態、法說會 |
| 理財網站（MoneyDJ、CMoney）| 個股分析、籌碼 |
| 法人研究報告 | 深度分析（需注意來源可靠性）|

## 注意事項

1. **時效性**：股市資訊變化快，確認資料日期
2. **來源可靠性**：優先採用官方公告與主流財經媒體
3. **避免偏誤**：同時搜尋正面與負面觀點
4. **免責聲明**：每次輸出都要提醒「不構成投資建議」
