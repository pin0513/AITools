# 基本面分析參考

## 核心財務指標

### 獲利能力
| 指標 | 說明 | 良好標準（參考）|
|------|------|----------------|
| EPS | 每股盈餘 | 正成長、穩定 |
| 毛利率 | (營收-成本)/營收 | 產業前段班 |
| 營業利益率 | 本業獲利能力 | 穩定或上升 |
| ROE | 股東權益報酬率 | >15% |
| ROA | 資產報酬率 | >8% |

### 成長性
| 指標 | 說明 |
|------|------|
| 營收年增率 | YoY 營收成長 |
| 營收月增率 | MoM 營收變化 |
| EPS 成長率 | 獲利成長趨勢 |

### 財務健全度
| 指標 | 說明 | 警戒標準 |
|------|------|---------|
| 負債比 | 負債/總資產 | >70% 需留意 |
| 流動比 | 流動資產/流動負債 | <100% 需留意 |
| 速動比 | (流動資產-存貨)/流動負債 | <80% 需留意 |
| 利息保障倍數 | EBIT/利息支出 | <3 需留意 |

## 估值指標

| 指標 | 說明 | 使用情境 |
|------|------|---------|
| 本益比 (P/E) | 股價/EPS | 穩定獲利公司 |
| 股價淨值比 (P/B) | 股價/每股淨值 | 資產股、金融股 |
| 本益成長比 (PEG) | P/E / 盈餘成長率 | 成長股 |
| EV/EBITDA | 企業價值/稅前息前折舊攤銷前利潤 | 跨國比較 |

## 分析重點

1. **趨勢比數字重要**：看連續 4-8 季的變化
2. **與同業比較**：單一數字無意義，要看產業位置
3. **質化因素**：經營團隊、競爭優勢、產業地位
4. **前瞻指標**：在手訂單、產能利用率、新客戶
