# 消息面分析參考

## 消息來源分類

### 官方來源（可信度高）
| 來源 | 內容 |
|------|------|
| 公開資訊觀測站 | 財報、重大訊息、股東會 |
| 證交所/櫃買中心 | 交易資訊、申購公告 |
| 公司官網 | 產品資訊、新聞稿 |
| 法說會簡報 | 營運展望、財務數據 |

### 財經媒體（需判斷）
| 來源 | 特性 |
|------|------|
| 經濟日報 | 產業新聞、法說會報導 |
| 工商時報 | 產業分析、企業動態 |
| 財訊/今周刊 | 深度報導、投資分析 |
| 鉅亨網 | 即時新聞、國際動態 |

### 需謹慎看待
- 社群媒體傳言
- 匿名論壇消息
- 付費投顧報告（可能有利益衝突）
- LINE 群組「明牌」

## 重大消息類型

### 利多消息
| 類型 | 範例 |
|------|------|
| 接單 | 取得大客戶訂單、在手訂單創高 |
| 擴產 | 新廠落成、產能擴充 |
| 新品 | 新產品上市、技術突破 |
| 轉投資 | 子公司獲利、IPO |
| 股利 | 配息優於預期 |
| 庫藏股 | 公司買回自家股票 |

### 利空消息
| 類型 | 範例 |
|------|------|
| 訂單流失 | 大客戶轉單、砍單 |
| 營運警訊 | 營收大幅衰退、虧損擴大 |
| 經營權 | 董監異動、經營權爭奪 |
| 法律風險 | 訴訟、違規裁罰 |
| 減資 | 彌補虧損減資 |
| 下市風險 | 財報難產、淨值為負 |

## 消息解讀原則

### 時效性
- 確認消息日期，舊聞可能已反映在股價

### 影響程度
- 一次性 vs 持續性影響
- 對營收/獲利的實際影響金額

### 市場預期
- 利多出盡：好消息出來反而下跌
- 利空出盡：壞消息出完反而上漲

### 資訊不對稱
- 內部人可能提前知道
- 股價異常波動可能暗示消息

## 法說會重點

法說會是獲取公司資訊的重要來源，關注：

1. **營運展望**：下季/下年度目標
2. **產能規劃**：擴產時程、資本支出
3. **客戶動態**：主要客戶佔比、新客戶開發
4. **毛利率展望**：成本壓力、定價能力
5. **產業觀點**：管理層對景氣的看法
