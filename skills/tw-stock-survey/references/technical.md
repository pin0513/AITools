# 技術面分析參考

## 基本技術指標

### 均線系統
| 均線 | 用途 |
|------|------|
| 5MA | 週線，短線趨勢 |
| 10MA | 雙週線，短線支撐壓力 |
| 20MA | 月線，中短期趨勢 |
| 60MA | 季線，中期趨勢 |
| 120MA | 半年線，中長期趨勢 |
| 240MA | 年線，長期趨勢 |

### 趨勢判斷
| 型態 | 說明 |
|------|------|
| 多頭排列 | 短均線在上、長均線在下 |
| 空頭排列 | 長均線在上、短均線在下 |
| 黃金交叉 | 短均線向上穿越長均線 |
| 死亡交叉 | 短均線向下穿越長均線 |

## 常用技術指標

| 指標 | 說明 | 應用 |
|------|------|------|
| KD | 隨機指標 | 超買(>80)超賣(<20) |
| RSI | 相對強弱指標 | 超買(>70)超賣(<30) |
| MACD | 趨勢動能指標 | 柱狀體翻正/負 |
| 布林通道 | 價格波動區間 | 突破上下軌 |

## 成交量分析

| 現象 | 可能意義 |
|------|---------|
| 價漲量增 | 多方趨勢確認 |
| 價漲量縮 | 追價力道不足 |
| 價跌量增 | 空方趨勢確認 |
| 價跌量縮 | 賣壓減輕 |
| 爆大量 | 可能是轉折點 |

## 支撐與壓力

### 常見支撐壓力來源
- 前波高低點
- 整數關卡（如 100、500）
- 均線位置
- 缺口位置
- 成交密集區

## 搜尋範本

```
{代號} 技術分析
{代號} K線 走勢
{代號} 均線 支撐 壓力
{代號} 股價 創高 創低
```

## 技術分析限制

1. **落後指標**：反映過去，不預測未來
2. **自我實現**：太多人看同樣指標可能失效
3. **需搭配基本面**：技術面無法判斷公司價值
4. **個股特性不同**：成交量小的股票技術面參考性低

## 注意事項

- 技術分析適合短線交易參考
- 長期投資應以基本面為主
- 不要只看單一指標
- 設定停損停利，控制風險
