# 籌碼面分析參考

## 三大法人

| 法人 | 特性 |
|------|------|
| 外資 | 資金量大、偏好權值股、影響指數 |
| 投信 | 追蹤績效、季底作帳、偏好中小型成長股 |
| 自營商 | 短線操作、避險部位 |

## 籌碼指標

### 法人動向
| 指標 | 說明 | 解讀 |
|------|------|------|
| 法人連買天數 | 連續買超天數 | 越長越強勢 |
| 法人買賣超金額 | 單日/累計買賣超 | 觀察趨勢 |
| 外資持股比例 | 外資持有佔總股數% | 高比例=關注度高 |

### 融資融券
| 指標 | 說明 | 解讀 |
|------|------|------|
| 融資餘額 | 借錢買股的總量 | 高=散戶看多 |
| 融券餘額 | 借券賣出的總量 | 高=市場看空 |
| 券資比 | 融券/融資 | 高=軋空機會 |
| 融資使用率 | 融資餘額/融資限額 | 高=追價力道弱 |

### 大戶持股
| 指標 | 說明 |
|------|------|
| 千張大戶持股比例 | 持有 1,000 張以上的股東比例 |
| 董監持股比例 | 內部人持股 |
| 質押比例 | 董監質押股票比例（高=警訊）|

## 搜尋範本

```
{代號} 三大法人 買賣超
{代號} 外資 持股
{代號} 融資 融券
{代號} 大戶 持股 集中度
{代號} 董監 質押
```

## 籌碼分析原則

### 多方訊號
- 外資、投信同步買超
- 融資減少、融券增加（籌碼沉澱）
- 大戶持股比例上升
- 董監持股穩定或增加

### 空方警訊
- 法人連續賣超
- 融資大增（散戶追高）
- 大戶持股下降
- 董監質押比例過高（>50%）

## 注意事項

1. 籌碼是「輔助指標」，需搭配基本面
2. 外資不一定對，可能是避險或 ETF 調整
3. 融資不一定錯，要看位階高低
4. 單日數據參考性低，看趨勢變化
