# 股票申購資訊參考

## 申購流程時程

```
競價拍賣（3天）→ 開標定價 → 公開申購（5天）→ 抽籤 → 撥券 → 掛牌
```

## 關鍵資訊項目

| 項目 | 說明 |
|------|------|
| 承銷價 | 申購價格（通常低於市價）|
| 競拍底價 | 競價拍賣最低價 |
| 申購期間 | 可提出申購的日期區間 |
| 抽籤日 | 公布中籤結果 |
| 撥券日 | 股票撥入帳戶 |
| 掛牌日 | 開始上市/上櫃交易 |

## 價差計算

```
預估價差 = (參考價 - 承銷價) × 1,000股
預估報酬率 = (參考價 - 承銷價) / 承銷價 × 100%

申購成本 = 承銷價 × 1,000 + 20（手續費）+ 50（郵寄費，若適用）
```

## 評估要點

### 值得申購的訊號
- 價差報酬率 > 20%
- 公司基本面良好（獲利穩定成長）
- 產業趨勢向上
- 在手訂單能見度高

### 風險警示
- 承銷價與市價接近（價差小）
- 公司尚未獲利
- 產業景氣下行
- 解禁賣壓（大股東閉鎖期結束）

## 申購小知識

| 項目 | 說明 |
|------|------|
| 申購張數 | 一般為 1 張（1,000股）|
| 每人限購 | 通常 1 張 |
| 手續費 | 20 元 |
| 中籤率 | 視申購人數，通常 1%-10% |
| 未中籤 | 款項退回，僅扣手續費 |
