## 黃金圈框架 (Golden Circle)

### Simon Sinek's Golden Circle

```
         ┌─────────┐
         │   WHY   │  ← 為什麼？(信念/目的)
         │  ─────  │
         │   HOW   │  ← 如何做？(獨特方法)
         │ ─────── │
         │   WHAT  │  ← 做什麼？(產品/服務)
         └─────────┘

大多數人：從外圈(What)往內說
優秀品牌：從內圈(Why)往外說
```

### 黃金圈工作表

```markdown
## 黃金圈分析：[產品/公司名稱]

### WHY - 為什麼存在？
> 我們的信念是什麼？為什麼這件事重要？

**核心信念**：
___

**激勵語句** (不談產品，只談理念)：
「我們相信 ___」

---

### HOW - 如何實現？
> 我們的獨特方法/差異化是什麼？

**獨特價值**：
1. ___
2. ___
3. ___

**差異化語句**：
「我們透過 ___ 來實現這個信念」

---

### WHAT - 做什麼？
> 具體的產品或服務是什麼？

**產品/服務**：
___

**功能語句**：
「我們提供 ___」

---

### 完整品牌故事

「我們相信 [WHY]。
我們透過 [HOW] 來實現這個信念。
我們的產品 [WHAT] 正是這個理念的體現。
想要一起 [CTA] 嗎？」

### 範例：Apollo Forms

**WHY**: 我們相信 HR 不該把時間浪費在表單庶務上
**HOW**: 我們透過智慧化表單引擎，讓繁瑣流程自動化
**WHAT**: Apollo Forms 是企業級表單管理平台

**完整故事**：
「我們相信 HR 的價值在於關懷人才，而非處理表單。
我們透過智慧化表單引擎，將繁瑣的行政流程自動化。
Apollo Forms 讓 HR 每週省下 10 小時，專注在真正重要的事。」
```

## Jobs to be Done (JTBD)

### JTBD 框架

```markdown
## JTBD 分析

### Job Statement 公式
> When [情境], I want to [動機], so I can [預期結果]

### 範例

**功能性 Job**：
「當我需要讓新進員工簽署勞動契約時，
我希望有一個線上簽署流程，
這樣我就不用列印、寄送、等待、歸檔。」

**情感性 Job**：
「當老闆問我員工資料在哪時，
我希望 3 秒內就能找到，
這樣我才不會顯得不專業。」

**社會性 Job**：
「當同業討論數位化程度時，
我希望我們公司不會落後，
這樣我才能維護部門的形象。」
```

```markdown
### ROI 計算

```
年度價值 = $___
年度成本 = $___


## 客戶訪談指南

### 訪談問題庫

```markdown
## 客戶訪談問題

### 背景問題 (5分鐘)
- 請簡單介紹您的角色和職責
- 您的團隊規模多大？
- 目前用什麼工具處理 ___？

### 痛點探索 (15分鐘)
- 在 ___ 流程中，最花時間的是什麼？
- 上次遇到 ___ 問題是什麼時候？後來怎麼解決？
- 如果可以改變一件事，會是什麼？
- 這個問題如果不解決，會有什麼影響？

### 需求驗證 (10分鐘)
- 如果有一個工具可以 ___，您會有興趣嗎？
- 這樣的解決方案值多少錢？
- 決定採購這類工具，會看哪些因素？
- 除了您，還有誰會參與決策？

### 收尾 (5分鐘)
- 還有什麼我們沒問到但您想分享的？
- 可以推薦其他適合訪談的人嗎？
```
