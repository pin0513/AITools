## 市場定價研究

### 定價分析框架

```markdown
## 定價策略分析

### 市場價格帶分布

```
價格
  ↑
高 │  ┌─────┐
   │  │ 高端 │ ← 企業級、客製化
   │  └─────┘
   │     ┌─────────┐
中 │     │  主流   │ ← 標準功能、中型企業
   │     └─────────┘
   │        ┌───────────┐
低 │        │   入門    │ ← 基礎功能、SMB
   │        └───────────┘
   └─────────────────────────→ 市場份額
```

### 定價模式比較

| 模式 | 優點 | 缺點 | 適用場景 |
|------|------|------|----------|
| **訂閱制 (SaaS)** | 穩定現金流、低進入門檻 | 需持續證明價值 | 軟體服務 |
| **按人頭計費** | 簡單易懂、可預測 | 大客戶成本高 | HR/協作工具 |
| **按用量計費** | 符合實際使用 | 收入不穩定 | API/雲端服務 |
| **階梯定價** | 滿足不同需求 | 選擇困難 | 通用 |
| **買斷制** | 一次性收入高 | 無持續收入 | 傳統軟體 |

### 價格敏感度分析

| 客群 | 價格敏感度 | 決策者 | 支付意願上限 |
|------|------------|--------|--------------|
| 大型企業 | 🟢低 | CIO/CFO | $___ /人/月 |
| 中型企業 | 🟡中 | 部門主管 | $___ /人/月 |
| 小型企業 | 🔴高 | 老闆 | $___ /人/月 |

### 定價策略建議

| 策略 | 說明 | 適用時機 |
|------|------|----------|
| 滲透定價 | 低價搶市占 | 新進入市場 |
| 撇脂定價 | 高價取利潤 | 技術領先、差異化明顯 |
| 競爭定價 | 對標競品 | 成熟市場 |
| 價值定價 | 依據價值而非成本 | 有明確 ROI |
```
